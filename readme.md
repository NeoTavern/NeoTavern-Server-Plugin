A server plugin of [NeoTavern-Frontend](https://github.com/NeoTavern/NeoTavern-Frontend) for [SillyTavern](https://docs.sillytavern.app/).

## Installation

1. Open a terminal in `{SillyTavern_Folder}/plugins`.
```bash
git clone https://github.com/NeoTavern/NeoTavern-Server-Plugin
```

2. Set `enableServerPlugins: true` in `{SillyTavern_Folder}/config.yaml`.
3. Restart the server.
4. Use with [NeoTavern-Frontend](https://github.com/NeoTavern/NeoTavern-Frontend)
